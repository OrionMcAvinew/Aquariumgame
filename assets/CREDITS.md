# Third-party assets

## Fish Pack — Kenney
- Files: `fish/fish_*.png`, `decor/background_*.png`, `decor/bubble_*.png`
- Author: Kenney (https://kenney.nl)
- Source: https://kenney.nl/assets/fish-pack
- License: Creative Commons Zero (CC0 1.0 Universal) — public domain.
  https://creativecommons.org/publicdomain/zero/1.0/

These assets are dedicated to the public domain. Attribution is not
required but is given here as a courtesy. In-game, several species reuse
these base shapes recolored at runtime (hue/saturation tint, original
shading preserved).

## Character model — Quaternius
- File: `models/character.glb` (rigged human with Walk/Idle/Run/etc. animations)
- Author: Quaternius (https://quaternius.com)
- License: Creative Commons Zero (CC0 1.0) — public domain.

Used for shopper NPCs (cloned per customer with its own animation mixer).

## Checkout counter — Meshy AI
- File: `models/counter.glb`
- Generated with Meshy AI (https://www.meshy.ai) by the project owner.

Vertex normals are computed at load time and a clean white PBR material is
applied so it reads as a modern retail checkout. A textured export can be
dropped in later in place of this geometry-only mesh.
